# Oculis

Oculis is an iOS application designed to enhance public transit accessibility, especially for the visually impaired in Singapore.

## Target Users

- Visually impaired commuters in Singapore
- Anyone who needs assistance navigating public transit

## Features

- **Real-time Bus Scanning**: Uses on-device machine learning to identify buses
- **Journey Planning**: Complete transit companion with live data from Singapore's Land Transport Authority (LTA)
- **Accessibility-First**: Designed with accessibility as a core priority
